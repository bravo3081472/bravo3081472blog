專案
  • 購物網站-蛋糕
    主要功能
    • 1. 撈取 API 商品資料，並根據 API 商品資料是否啟用來決定商品是否上架
    • 2. 購買商品可以加入優惠碼決定是否打折
    • 3. 將商品加入購物車，購物車結帳後會列出清單填寫收寄人資訊
    • 4. 根據帳號系統管控是否加入管理者頁面
    • 5. 管理者頁面可以新增、編輯、刪除商品跟優惠碼，並查閱所有的訂單資訊
    開發部分
    • 資料庫：主要是使用六角學院提供的 RESTful API，並使用 AJAX 串接
    • CSS：主要使用 Bootstrap，並以此為基礎對於細節及 RWD 加上自己寫的 SCSS
    • 為了跟 Bootstrap 命名分開，CSS 命名使用 BEM 方式命名
    • JavaScript：使用 Vue-Cli 網站架構，框架使用 Vue.js，路由是使用 Vue-Router
    • 註解部分加入 JSdoc
    • 獨立開發
    • 開發時間：1~2個月，中間有重構過網站，主要是增加新功能並將分散的 CSS 集中管理
